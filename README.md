## 📌 Описание

**GeoTracing** — это инструмент для численного моделирования движения частиц в искривлённом пространстве-времени. Библиотека позволяет:
- Интегрировать геодезические уравнения (в Гамильтоновом формализме) для массивных и безмассовых частиц.
- Выполнять трассировку лучей для визуализации релятивистских эффектов (гравитационное линзирование, тень чёрной дыры, кротовые норы).
- Работать с различными метриками пространства-времени (Шварцшильда, Керра, Эллиса–Бронникова и др.).

Библиотека реализована на Python с использованием **OpenCL** с оберткой **PyOpenCL** для ускорения вычислений на GPU/CPU.

## Физика и математика моделирования

Библиотека **GeoTracing** реализует численное моделирование динамики частиц и света в искривлённом пространстве-времени в рамках **Общей Теории Относительности (ОТО)**. Основу моделирования составляют **геодезические уравнения**, описывающие движение свободных частиц и световых лучей в заданной метрике.

### Гамильтонов формализм

Для повышения вычислительной эффективности и устойчивости интегрирования используется **гамильтонов формализм**. Вместо уравнений второго порядка с символами Кристоффеля используется система уравнений первого порядка:

**Гамильтониан** для частицы/фотона:
\[
H(x^\alpha, p_\beta) = \frac{1}{2} g^{\mu\nu}(x^\alpha) p_\mu p_\nu
\]

**Уравнения Гамильтона**:
\[
\frac{dx^\alpha}{d\lambda} = \frac{\partial H}{\partial p_\alpha}, \quad
\frac{dp_\alpha}{d\lambda} = -\frac{\partial H}{\partial x^\alpha}
\]

где:
- \( x^\alpha \) — координаты в пространстве-времени
- \( p_\alpha \) — канонические импульсы
- \( \lambda \) — аффинный параметр (собственное время для массивных частиц)
- \( g^{\mu\nu} \) — контравариантные компоненты метрического тензора

### Пример: метрика ℝ × S²

Для метрики вида:
\[
ds^2 = -c^2 dt^2 + dw^2 + R^2(d\theta^2 + \sin^2\theta d\phi^2)
\]

**Гамильтониан** принимает вид:
\[
H = \frac{1}{2}\left(-\frac{p_t^2}{c^2} + p_w^2 + \frac{p_\theta^2}{R^2} + \frac{p_\phi^2}{R^2\sin^2\theta}\right)
\]

**Уравнения движения**:
\[
\begin{aligned}
\frac{dt}{d\lambda} &= -\frac{p_t}{c^2}, \quad \frac{dw}{d\lambda} = p_w, \quad \frac{d\theta}{d\lambda} = \frac{p_\theta}{R^2}, \quad \frac{d\phi}{d\lambda} = \frac{p_\phi}{R^2\sin^2\theta} \\
\frac{dp_t}{d\lambda} &= 0, \quad \frac{dp_w}{d\lambda} = 0, \quad \frac{dp_\theta}{d\lambda} = \frac{p_\phi^2 \cos\theta}{R^2\sin^3\theta}, \quad \frac{dp_\phi}{d\lambda} = 0
\end{aligned}
\]

### Трассировка лучей для визуализации

Для визуализации релятивистских эффектов (гравитационное линзирование, тени чёрных дыр, кротовые норы) используется **обратная трассировка лучей**:

1. **Карта небесных сфер**: Каждому направлению на локальном небе камеры ставится в соответствие точка на одной из двух небесных сфер червоточины (см. рис. 4 статьи).

2. **Интегрирование уравнений**: Лучи интегрируются назад во времени от камеры к небесным сферам с использованием уравнений геодезических.

3. **Построение изображения**: Интенсивность и цвет каждого пикселя определяются интегрированием по лучу с учётом источников на небесных сферах.

### Численная реализация

В библиотеке используется **гибридный подход**:
- **Python**: Высокоуровневый интерфейс, задание метрик, управление вычислениями
- **OpenCL**: Высокопроизводительное интегрирование уравнений на GPU/CPU

**Ключевые особенности**:
- Поддержка произвольных метрик пространства-времени
- Автоматическое вычисление символов Кристоффеля и компонент метрики
- Адаптивные методы интегрирования (Рунге-Кутта)
- Обработка особенностей координат (например, полюсов сферических координат)

### Примеры метрик

Библиотека поддерживает следующие метрики:

**1. Метрика Минковского** (плоское пространство-время):
\[
ds^2 = -dt^2 + dx^2 + dy^2 + dz^2
\]

**2. Метрика Шварцшильда** (невращающаяся чёрная дыра):
\[
ds^2 = -\left(1 - \frac{2GM}{c^2 r}\right) dt^2 + \left(1 - \frac{2GM}{c^2 r}\right)^{-1} dr^2 + r^2 d\Omega^2
\]

**3. Метрика Керра** (вращающаяся чёрная дыра):
\[
ds^2 = -\left(1 - \frac{2GMr}{\Sigma}\right) dt^2 - \frac{4GMar\sin^2\theta}{\Sigma} dt d\phi + \frac{\Sigma}{\Delta} dr^2 + \Sigma d\theta^2 + \left(r^2 + a^2 + \frac{2GMa^2 r \sin^2\theta}{\Sigma}\right) \sin^2\theta d\phi^2
\]

**4. Метрика червоточины Эллиса**:
\[
ds^2 = -dt^2 + d\ell^2 + (\ell^2 + a^2)(d\theta^2 + \sin^2\theta d\phi^2)
\]

**5. Метрика червоточины Dneg** (использована в "Интерстелларе"):
\[
ds^2 = -(1 + 2\Phi)dt^2 + d\ell^2 + r^2(\ell)(d\theta^2 + \sin^2\theta d\phi^2)
\]
где \( r(\ell) \) задаётся параметрически для обеспечения плавных переходов.

### Условная нормировка

Для различных типов геодезических выполняется условие:
\[
g_{\mu\nu}\frac{dx^\mu}{d\lambda}\frac{dx^\nu}{d\lambda} = \epsilon
\]
где:
- \( \epsilon = -1 \) — времениподобные геодезические (массивные частицы)
- \( \epsilon = 0 \) — светоподобные геодезические (фотоны)
- \( \epsilon = +1 \) — пространственноподобные геодезические

### Сохраняющиеся величины

Благодаря симметриям метрик сохраняются:
- **Энергия**: \( E = -p_t \)
- **Угловой момент**: \( L_z = p_\phi \)
- **Полный угловой момент**: \( L^2 = p_\theta^2 + \frac{p_\phi^2}{\sin^2\theta} \)

Эти сохраняющиеся величины используются для проверки точности численного интегрирования и анализа траекторий.

### Практическое применение

Методы, реализованные в библиотеке, позволяют:
1. Моделировать траектории частиц в окрестностях чёрных дыр
2. Визуализировать гравитационное линзирование
3. Строить изображения аккреционных дисков
4. Моделировать прохождение через кротовые норы
5. Исследовать влияние параметров метрики на наблюдаемые эффекты

Библиотека предоставляет инструменты для исследований и образования в области релятивистской астрофизики и компьютерной графики.
